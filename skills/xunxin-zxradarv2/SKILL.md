# 风控-综合指数V2 · ZX-RadarV2_1 2.1.0

## SKILL 定位

本 SKILL 是4个产品中数据维度最丰富的综合评估工具，适用于需要建模或精细化风险策略的场景。三个子报告各有侧重：

- **申请雷达（apply_report_detail）**：捕捉申请行为特征——申请准入分（A22160001）反映近期申请行为的风险程度，高频申请是资金紧迫的强信号
- **行为雷达（behavior_report_detail）**：还原用户借还款全链路历史——贷款行为分（B22170001）综合了近24个月的贷款笔数、逾期笔数、履约金额等54个细粒度字段
- **信用现状（current_report_detail）**：估算用户当前信用额度上限——网贷授信额度（C22180001）和消金建议授信额度（C22180011）可直接用于本次授信参考

相比其他产品，综合指数V2 字段最多、维度最全，是入模和策略调优的首选数据源，KS/AUC 指标优于单一维度产品。

## MCP 依赖

- 必选：`mandao-company`（`getXunxinZxRadarV2`）

## 通用执行原则

**第一，三个子报告需分别解读，不可混淆。** `apply_report_detail`、`behavior_report_detail`、`current_report_detail` 各自独立，对应申请时点、历史行为、当前状态三个维度，须分别评估后综合。

**第二，申请准入分（A22160001）是快速决策的核心单指标。** 分值范围1-1000，分值越低风险越高。在策略评估中，该分通常作为第一层切分阈值。

**第三，近期查询频率是申请欺诈的核心信号。** `A22160008`（近1月查询笔数）和 `A22160009`（近3月查询笔数）短期内骤升，说明申请人正在密集申请信贷，是资金链断裂前的典型行为特征。

**第四，逾期笔数需按时间窗口分层解读。** `B22170025`（近6月M0+逾期）→ `B22170026`（近12月）→ `B22170027`（近24月）反映逾期的时间衰减，近期逾期权重显著高于远期。

**第五，授信额度建议值（C22180001/C22180011）须结合置信度使用。** `C22180002`（网贷额度置信度）和 `C22180012`（消金额度置信度）低于60时，额度建议值参考价值有限，需人工判断。

## 工作流

### Step 1：调用 MCP 获取数据

```
mcp__mandao-company__getXunxinZxRadarV2
参数：{ "idNo": "<身份证号>", "idName": "<姓名>", "phoneNo": "<手机号>" }
```

### Step 2：判断业务返回码

| data.code | 处理方式                                                 |
| --------- | -------------------------------------------------------- |
| `0`       | 查询命中，进入 Step 3-5 解读三个子报告                   |
| `1`       | 未命中，用户无三方支付借贷数据，建议配合其他产品综合判断 |
| `9`       | 系统异常，提示重新查询                                   |

### Step 3：申请雷达解读（apply_report_detail）

| 字段        | 说明                     | 风险解读                                     |
| ----------- | ------------------------ | -------------------------------------------- |
| `A22160001` | 申请准入分（1-1000）     | **< 400 高风险，400-600 中等，> 600 低风险** |
| `A22160002` | 申请准入置信度（50-100） | < 60 时分数参考价值降低                      |
| `A22160003` | 申请命中机构数           | 机构数越多，多头申请越严重                   |
| `A22160008` | 近1月查询笔数            | ≥ 5 为高频申请预警                           |
| `A22160009` | 近3月查询笔数            | 结合近1月判断申请趋势                        |
| `A22160010` | 近6月查询笔数            | 基准参照                                     |

### Step 4：行为雷达解读（behavior_report_detail）

**核心评分**：

| 字段        | 说明                     | 风险解读                |
| ----------- | ------------------------ | ----------------------- |
| `B22170001` | 贷款行为分（1-1000）     | 分值越低历史行为越差    |
| `B22170051` | 贷款行为置信度（50-100） | < 60 时分数参考价值降低 |
| `B22170034` | 正常还款订单占比（%）    | < 70% 为还款质量偏低    |

**逾期笔数（重点关注近6/12个月）**：

| 字段        | 说明                                 |
| ----------- | ------------------------------------ |
| `B22170025` | 近6月M0+逾期笔数（当期逾期，最严重） |
| `B22170026` | 近12月M0+逾期笔数                    |
| `B22170028` | 近6月M1+逾期笔数（逾期1个账期以上）  |
| `B22170029` | 近12月M1+逾期笔数                    |

> M0+ 表示当期及以上逾期；M1+ 表示跨期逾期，债务状态更严重

**借款规模**：`B22170002`-`B22170011` 反映近1/3/6/12/24个月贷款笔数和金额，可判断借款习惯与还款能力匹配度。

**扣款失败**：`B22170035`-`B22170039`（近1-24月失败扣款笔数），持续失败扣款是账户资金紧张的直接信号。

### Step 5：信用现状解读（current_report_detail）

| 字段        | 说明                     | 使用方式                |
| ----------- | ------------------------ | ----------------------- |
| `C22180001` | 网贷建议授信额度（元）   | 本次网贷授信上限参考    |
| `C22180002` | 网贷额度置信度（50-100） | < 60 时额度建议仅供参考 |
| `C22180011` | 消金建议授信额度（元）   | 本次消金授信上限参考    |
| `C22180012` | 消金额度置信度（50-100） | < 60 时额度建议仅供参考 |
| `C22180003` | 网络贷款机构数           | 当前在贷机构数量        |
| `C22180007` | 消金贷款机构数           | 当前消金在贷机构数量    |

### Step 6：综合风险定级

| 风险等级        | 触发条件                                                   | 准入建议                           |
| --------------- | ---------------------------------------------------------- | ---------------------------------- |
| 🔴 **高风险**   | 申请准入分 < 400，或近6月M0+逾期 ≥ 3笔，或贷款行为分 < 400 | **拒绝**                           |
| 🟠 **中高风险** | 申请准入分 400-500，或近1月查询 ≥ 5次，或正常还款率 < 70%  | **人工复核**                       |
| 🟡 **中风险**   | 申请准入分 500-600，历史有逾期但近期改善，置信度偏低       | **审慎准入，参考建议额度降幅授信** |
| 🟢 **低风险**   | 申请准入分 > 600，贷款行为分 > 600，还款率高，近期无逾期   | **正常准入，参考建议额度授信**     |

## 输出模板

- **章节 1：风险结论 · Decision Pack**（综合风险等级 + 申请准入分 + 贷款行为分 + 授信建议）
- **章节 2：申请雷达分析**（申请行为分、近期查询频率、命中机构数）
- **章节 3：行为雷达分析**（借款规模趋势、逾期笔数分层、还款质量、失败扣款）
- **章节 4：信用现状**（当前建议授信额度、在贷机构数、置信度说明）
- **章节 5：综合风险评估与授信建议**（三个维度交叉验证结论）
- **章节 6：数据来源与查询时间戳**

## 参数

- `--idNo`：身份证号（必填），末位 X 须大写
- `--idName`：姓名（必填）
- `--phoneNo`：手机号（可选，提升命中率）
- `--format md|json`：输出格式，默认 md

## 边界与免责

本 SKILL 字段数量多（50+），解读需具备一定的信贷风控背景知识。各字段的具体阈值应结合机构自身用户群体的数据分布进行回测标定，本 SKILL 提供的阈值为通用参考值。

建议授信额度（C22180001/C22180011）由模型估算，非正式授信决策，须结合机构内部评分卡和风控政策最终确定。
