## 工具列表（共 4 个）

> 下述 4 个工具的入参结构一致：`idNo` / `idName` / `phoneNo`。

### 1) `getXunxinZxRadarV2`

**讯信-综合指数V2**（ZX-RadarV2_1 / 2.1.0）。该产品是基于三方支付数据开发的用于评估用户信贷全流程的综合类产品，通过用户在各类信贷平台上的申请、借款、履约、逾期等信贷行为，及时分析贷款人的借贷全链路行为；适用于信贷领域贷前审批、风险定价及授信额度管理、贷中监测等风控环节；建议入模、策略评估时使用，关键评估指标：KS、AUC等。

**参数**

| 参数名    | 类型     | 说明     |
| --------- | -------- | -------- |
| `idNo`    | `string` | 身份证号 |
| `idName`  | `string` | 姓名     |
| `phoneNo` | `string` | 手机号   |

### 2) `getXunxinQjdaV130`

**讯信-全景指数**（QJDA / 1.3.0）。该产品是基于三方支付数据开发的分析用户信贷逾期和共债状态的综合产品，通过分析用户在不同平台的逾期借贷和借贷重叠行为，反馈用户的逾期情况和共债情况，帮助商户控制信贷风险，助力企业规避风险用户；适用于信贷业务的贷前申请、授信环节；建议注重策略评估时使用，关键评估指标：击中率、拒绝率、风险提升度等。

**参数**

| 参数名    | 类型     | 说明     |
| --------- | -------- | -------- |
| `idNo`    | `string` | 身份证号 |
| `idName`  | `string` | 姓名     |
| `phoneNo` | `string` | 手机号   |

### 3) `getXunxinFmlhV140`

**讯信-履约指数**（FMLH / 1.4.0）。该产品基于三方支付数据，用于评估用户近期是否正常履约；适用于信贷业务的贷前审批环节；建议注重策略评估时使用，关键评估指标：击中率、拒绝率、风险提升度等。

**参数**

| 参数名    | 类型     | 说明     |
| --------- | -------- | -------- |
| `idNo`    | `string` | 身份证号 |
| `idName`  | `string` | 姓名     |
| `phoneNo` | `string` | 手机号   |

### 4) `getXunxinQjtzV140`

**讯信-信用探查指数**（QJTZ / 1.4.0）。该产品基于三方支付数据，用于评估用户逾期和履约情况；适用于信贷业务的贷前审批环节；建议注重策略评估时使用，关键评估指标：击中率、拒绝率、风险提升度等。
